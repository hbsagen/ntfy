# Ads and ad platforms

Method legend: webhook, script, polling, email, ci

- Google Ads — Possible: yes. Method: script/polling. Quirks: no outbound webhooks; use API polling + cron. Docs: https://developers.google.com/google-ads/api
- Google Ad Manager — Possible: yes. Method: script/polling. Quirks: API polling. Docs: https://developers.google.com/ad-manager/api
- Meta Ads (Facebook/Instagram) — Possible: yes. Method: webhook. Quirks: use Marketing API + webhooks. Docs: https://developers.facebook.com/
- LinkedIn Ads — Possible: yes. Method: script/polling. Quirks: no outbound webhooks; use API polling. Docs: https://learn.microsoft.com/linkedin/marketing/
- TikTok Ads — Possible: yes. Method: webhook. Quirks: webhooks for events. Docs: https://ads.tiktok.com/marketing_api/docs
- Snapchat Ads — Possible: yes. Method: script/polling. Quirks: API polling. Docs: https://marketingapi.snapchat.com/
- Pinterest Ads — Possible: yes. Method: webhook/script. Quirks: limited webhooks; use API polling. Docs: https://developers.pinterest.com/docs/redoc/
- Reddit Ads — Possible: yes. Method: script/polling. Quirks: API polling. Docs: https://ads-api.reddit.com/docs/
- Microsoft Advertising (Bing Ads) — Possible: yes. Method: script/polling. Quirks: API polling. Docs: https://learn.microsoft.com/advertising/
- Amazon Ads — Possible: yes. Method: script/polling. Quirks: API polling. Docs: https://advertising.amazon.com/API/docs
